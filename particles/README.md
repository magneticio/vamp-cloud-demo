# Particles Demo

## Create the Demo Autoscaler Service Account
```sh
kubectl apply -f demo-autoscaler-sa.yaml
```

## Deploy Version 1.0.0
```sh
kubectl apply -f particles-v1.0.0.yaml
```

## Deploy Version 1.0.1
```sh
kubectl apply -f particles-v1.0.1.yaml
```
